Place your local brand font files here to allow the PDF generator and site to use them directly.

Preferred files:
- SF-Pro-Display-Regular.ttf
- SF-Pro-Display-Bold.ttf
- SF-Pro-Text-Regular.ttf
- SF-Pro-Text-Medium.ttf

The PDF generator already searches `assets/fonts` and common macOS font directories. If these files are not present, it falls back to the best available system font.
